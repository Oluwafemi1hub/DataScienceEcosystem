# DataScienceEcosystem
This exercise shows how to complete a Jupyter Notebook, save the progress, and share it on GitHub for collaboration and version control.
Awesome—here’s a clear, copy-paste, step-by-step for every exercise. I’ll assume you’re using **JupyterLite (Try Jupyter → Lab)**, but this works the same in Colab/Anaconda.

---

# Quick setup

1. Open JupyterLite: [https://jupyter.org/try-jupyter/lab/](https://jupyter.org/try-jupyter/lab/)
2. **File → New → Notebook → Python**
3. **File → Save Notebook As…** name it exactly: **`DataScienceEcosystem.ipynb`**

## 12) Save & share on GitHub

1. **File → Save**
2. **File → Download** → it saves **DataScienceEcosystem.ipynb** to your computer.
3. Go to **github.com** → log in → **New** repository → name it (public).
4. Click **Add file → Upload files** → drag in `DataScienceEcosystem.ipynb` → **Commit**.
5. Open the file in GitHub and copy the **URL** (you’ll submit this link).
